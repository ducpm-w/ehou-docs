# EHOU Docs

Tài liệu học tập cá nhân — static site, deploy on Vercel.

## Thêm môn học mới

1. Copy `subjects/web-programming.html` → đổi tên
2. Sửa nội dung, màu accent (`--subject-accent`), icon
3. Thêm card vào `index.html`
4. `git add . && git commit -m "add: môn [tên]" && git push`

Vercel tự deploy trong vòng 30 giây.
