Lập trình Web

Học cách xây dựng website từ nền tảng: cấu trúc HTML, trang trí CSS, và tương tác JavaScript. Từ trang tĩnh đến ứng dụng web động.

Giới thiệu môn học

Lập trình Web là môn học nền tảng cho ngành CNTT, cung cấp kỹ năng xây dựng giao diện và ứng dụng trên nền web — môi trường phổ biến nhất hiện nay.

Mục tiêu Sau môn học này, bạn có thể tự xây dựng một website hoàn chỉnh từ thiết kế đến triển khai.

HTML cơ bản

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu định nghĩa cấu trúc của trang web. Trình duyệt đọc HTML và hiển thị nội dung tương ứng.

Cấu trúc trang HTML

<!DOCTYPE html>
<html lang="vi">
  <head>
    <meta charset="UTF-8" />
    <title>Tiêu đề trang</title>
  </head>
  <body>
    <h1>Xin chào thế giới!</h1>
    <p>Đây là đoạn văn đầu tiên.</p>
  </body>
</html>

Các thẻ quan trọng

  • <h1> — <h6>: Tiêu đề, từ lớn đến nhỏ
  • <p>: Đoạn văn bản
  • <a href="">: Liên kết
  • <img src="" alt="">: Hình ảnh
  • <ul> / <ol> / <li>: Danh sách
  • <div> / <span>: Container tổng quát
  • <input>, <form>, <button>: Form nhập liệu
Ghi chú HTML5 giới thiệu các thẻ semantic như <header>, <nav>, <main>, <section>, <footer> — giúp code có ý nghĩa hơn và tốt cho SEO.

CSS & Styling

CSS (Cascading Style Sheets) kiểm soát giao diện: màu sắc, kích thước, vị trí, và animation của các phần tử HTML.

Cú pháp CSS

selector {
  property: value;
}

/* Ví dụ */
h1 {
  color: #6c5ce7;
  font-size: 2rem;
  font-weight: 700;
}

Box Model

Mọi phần tử HTML đều là một "hộp" với 4 lớp từ trong ra ngoài:

LớpMô tả
ContentNội dung thực (text, image)
PaddingKhoảng cách từ content đến border
BorderĐường viền xung quanh
MarginKhoảng cách từ border đến phần tử bên ngoài

Flexbox — Layout hiện đại

.container {
  display: flex;
  justify-content: center;   /* ngang */
  align-items: center;       /* dọc */
  gap: 1rem;
}
Mẹo Dùng display: flex để căn giữa — đây là cách phổ biến và dễ nhất hiện nay.

JavaScript

JavaScript làm cho trang web tương tác — xử lý sự kiện click, validate form, gọi API, thay đổi nội dung mà không cần reload trang.

Khai báo biến

const name = "Minh Đức";   // hằng số, không đổi
let age = 21;               // biến, có thể thay đổi
var old = true;             // cũ, tránh dùng

Xử lý sự kiện DOM

const btn = document.getElementById('my-btn');

btn.addEventListener('click', () => {
  alert('Bạn vừa click!');
});

Fetch API — Gọi dữ liệu

async function getData() {
  const res = await fetch('https://api.example.com/data');
  const json = await res.json();
  console.log(json);
}
Lưu ý Luôn dùng const mặc định, chỉ dùng let khi cần reassign. Không dùng var.

Responsive Design

Responsive design đảm bảo website hiển thị đẹp trên mọi thiết bị — từ điện thoại (320px) đến màn hình lớn (1920px+).

Media Queries

/* Mobile first — viết cho mobile trước */
.card {
  width: 100%;
}

/* Tablet trở lên */
@media (min-width: 640px) {
  .card { width: 50%; }
}

/* Desktop */
@media (min-width: 1024px) {
  .card { width: 33.33%; }
}

Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0" />
Nguyên tắc Mobile-first: Thiết kế cho màn hình nhỏ trước, sau đó mở rộng dần cho màn hình lớn hơn bằng media queries.
